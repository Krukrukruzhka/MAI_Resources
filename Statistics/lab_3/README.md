# Анализ корреляционных зависимостей и влияние выбросов

## 1. Постановка задачи

**Цель исследования:** провести сравнительный анализ различных типов корреляционных зависимостей и исследовать влияние выбросов на коэффициенты корреляции.

**Основные задачи:**
1. Сгенерировать две независимые выборки объёмом 1000: X ~ N(0,1) и Z ~ N(5,2)
2. Вычислить Y тремя способами для каждой пары (x,z):
   - Линейная зависимость: Y = aX + (1-a)Z для a ∈ {0, 0.25, 0.5, 0.75, 1}
   - Кубическая зависимость: Y = 0.05X³ + 4Z
   - Квадратичная зависимость: Y = X² + Z
3. Добавить выбросы к зависимостям и проанализировать их влияние
4. Вычислить коэффициенты корреляции Пирсона, Спирмана и Кендалла
5. Проверить гипотезы о равенстве коэффициентов корреляции нулю на уровне значимости 0.05

## 2. Методы исследования

### 2.1 Генерация данных
- **Выборка X:** нормальное распределение N(0,1)
- **Выборка Z:** нормальное распределение N(5,2)
- **Объём выборки:** 1000 наблюдений

### 2.2 Типы зависимостей
1. **Линейная:** Y = aX + (1-a)Z
2. **Кубическая:** Y = 0.05X³ + 4Z  
3. **Квадратичная:** Y = X² + Z

### 2.3 Коэффициенты корреляции
- **Пирсон:** мера линейной связи
- **Спирман:** мера монотонной связи
- **Кендалл:** мера ранговой корреляции

### 2.4 Статистические тесты
- Проверка гипотезы H₀: ρ = 0
- Уровень значимости: α = 0.05

## 3. Результаты и выводы

### 3.1 Основные результаты корреляционного анализа

**Линейные зависимости Y = aX + (1-a)Z:**

| Коэффициент | a=0 | a=0.25 | a=0.5 | a=0.75 | a=1 |
|-------------|-----|--------|-------|--------|-----|
| **Пирсон** | -0.040 | 0.122 | 0.411 | 0.820 | 1.000 |
| **Спирман** | -0.064 | 0.091 | 0.379 | 0.810 | 1.000 |
| **Кендалл** | -0.044 | 0.060 | 0.258 | 0.613 | 1.000 |

**Статистическая значимость (α = 0.05):**
- При a=0 (Y=Z): корреляция должна быть незначимой, но из-за большого объема выборки может наблюдаться ложное обнаружение
- При a≥0.25: все коэффициенты статистически значимы, однако для слабых корреляций (например, a=0.25, r=0.122) это может быть следствием большого объема выборки

**Практическая значимость:**
- Корреляция 0.122 (a=0.25) считается слабой и объясняет только 1.5% дисперсии
- Статистическая значимость не означает практическую важность связи

**Нелинейные зависимости:**

| Коэффициент | Кубическая | Квадратичная |
|-------------|------------|--------------|
| **Пирсон** | -0.022 | 0.028 |
| **Спирман** | -0.050 | -0.022 |
| **Кендалл** | -0.035 | -0.019 |

**Статистическая значимость:** для нелинейных зависимостей гипотеза H₀ не отвергается ни для одного коэффициента

### 3.2 Методологические замечания

**Проблема больших выборок:** При объеме выборки n=1000 даже очень слабые случайные корреляции могут стать статистически значимыми (p < 0.05). Это объясняет, почему для Y=Z (где корреляция должна быть нулевой) мы получаем статистическую значимость.

**Различие статистической и практической значимости:**
- **Статистическая значимость:** определяет, существует ли связь между переменными вообще
- **Практическая значимость:** определяет, насколько сильная эта связь на практике
- **Пример:** Корреляция 0.122 статистически значима (p=0.0001), но практически слабая (объясняет только 1.5% дисперсии)

**Корректная интерпретация:**
- Для Y=Z: несмотря на статистическую значимость, корреляция является случайной (ложное обнаружение)
- Для других зависимостей: статистическая значимость отражает реальную связь, но не всегда означает практическую важность
- **Рекомендация:** учитывать как p-значение, так и величину коэффициента корреляции

### 3.2 Влияние выбросов на коэффициенты корреляции

**Коэффициент Пирсона:**
- Наиболее чувствителен к выбросам
- Среднее изменение коэффициента: 35.1%
- Максимальное изменение: 199.5% (кубическая зависимость)

**Коэффициенты Спирмана и Кендалла:**
- Более устойчивы к выбросам
- Среднее изменение: около 0.6%
- Сохраняют стабильность даже при наличии выбросов

### 3.3 Проверка гипотез

На уровне значимости α = 0.05:
- Для линейных зависимостей с a ≥ 0.25 гипотеза H₀: ρ = 0 отвергается
- Для нелинейных зависимостей гипотеза H₀ не отвергается
- Выбросы не влияют на статистическую значимость коэффициентов Спирмана и Кендалла

### 3.4 Основные выводы

1. **Тип зависимости существенно влияет на корреляцию:** линейные зависимости показывают более сильные корреляции по сравнению с нелинейными.

2. **Коэффициент Пирсона эффективен для линейных зависимостей,** но сильно подвержен влиянию выбросов.

3. **Коэффициенты Спирмана и Кендалла более устойчивы** к выбросам и лучше подходят для анализа данных с аномалиями.

4. **Нелинейные зависимости (кубическая и квадратичная)** не демонстрируют значимой корреляции с X, что ожидаемо для сложных функциональных связей.
